## 2. Description of the Service

Agriculture RMS is a web-based system that allows:

- **Visitors** to view public information, news, and advisories related to agricultural programs in Claveria
- **Registered users** to create accounts, submit and manage enrollment or claim data (e.g., farmer profiles, livelihood information), and access features available to their role
- **Authorized staff** to manage farmer records, enrollments, news, and system administration

The system is provided to support the DA Claveria’s mandate to manage farmer records, crop data, and agricultural programs efficiently.
